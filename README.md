## 外星人入侵：使用 Python 开发游戏

在项目“外星人入侵”（第 12 章~第 14 章）中，你将使用 Pygame 包来开发一款 2D 游戏。它在玩家每消灭一群向下移动的外星人后，将玩家提高一个等级。等级越高，游戏的节奏越快，难度越大。完成这个项目后，你将获得自己动手使用 Pygame 开发 2D 游戏所需的技能。
